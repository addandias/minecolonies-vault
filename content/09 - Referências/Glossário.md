# Glossário

| Termo oficial | Uso no Vault | Significado |
|---|---|---|
| ferramenta de construção (*Build Tool*) | Ferramenta de construção | Posiciona prévias e gerencia esquemas |
| construtor | Construtor | Executa ordens de construção, melhoria e reparo |
| Building Hut | Bloco da construção | Bloco funcional usado para criar e administrar um edifício |
| Citizen | Cidadão | Habitante da colônia |
| Colony | Colônia | Área administrada pela Prefeitura |
| entregador | Entregador | Transporta pedidos entre construções |
| Salão de Refeições | Salão de Refeições | Distribui comida aos cidadãos; antigo Restaurant |
| campo | Campo | Área agrícola vinculada à Farmer’s Hut |
| Happiness | Felicidade | Indicador global das condições da colônia |
| Huscarl | Huscarl | guarda corpo a corpo que usa machado |
| Hut Level | Nível da construção | Progressão funcional de 1 a 5 |
| Estoque mínimo (*Minimum Stock*) | Estoque mínimo | Quantidade que uma construção tenta manter disponível |
| atirador | Atirador | guarda de longo alcance que usa besta |
| Prioridade de coleta (*Pickup Priority*) | Prioridade de retirada | Ordem de atendimento dos entregadores |
| Rack | Estante | Armazenamento interno das construções |
| Recipe | Receita | Transformação ensinada a uma oficina |
| Request | Pedido | Solicitação de item, ferramenta, alimento ou entrega |
| Research | Pesquisa | Melhoria ou desbloqueio desenvolvido na University |
| Residência | Residência | Moradia dos cidadãos; antiga Citizen’s Hut |
| Schema | Esquema | Projeto visual de uma construção |
| Prefeitura | Prefeitura | Centro administrativo da colônia |
| Armazém | Armazém | Centro da rede logística |
| Work Order | Ordem de trabalho | Construção, melhoria ou reparo atribuído a um construtor |

## Convenção de nomes

O Vault mantém o nome oficial em inglês no título e uma tradução descritiva em português. Nomes antigos permanecem como aliases quando ajudam quem joga em versões ou traduções diferentes.
