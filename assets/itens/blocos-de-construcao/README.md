# Blocos das construções

Esta área guarda os ícones dos blocos usados para criar e administrar as construções do MineColonies.

```text
blocos-de-construcao/
└── <identificador-do-item>/
    ├── principal.png
    └── variantes/
        └── <variante>.png
```

- `principal.png`: ícone canônico exibido na ficha lateral da construção.
- `variantes/`: use apenas quando o próprio mod tiver representações visuais distintas.
- A pasta preserva o identificador original, como `blockhutbuilder` ou `blockhuttownhall`.
- `Stable` e `Gatehouse` permanecem sem ícone lateral porque o snapshot indicado não contém seus blocos de construção.
