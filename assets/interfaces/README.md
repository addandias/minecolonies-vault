# Interfaces das construções

As capturas da interface do bloco de cada construção ficam em uma pasta própria.

```text
interfaces/
└── construcoes/
    └── <construcao>/
        ├── principal.png
        ├── configuracoes.png
        └── <aba-especifica>.png
```

- Use o identificador oficial normalizado da construção, em minúsculas e com hífen.
- Traduza o nome do arquivo para descrever a função da aba, sem alterar textos que aparecem dentro da captura.
- Não presuma que uma captura da wiki atual representa a interface da versão 1.20.1; informe a versão ou marque-a como referência atual quando não for possível confirmá-la.
- Na página, apresente primeiro as abas essenciais e mantenha telas complementares em uma galeria recolhível quando houver muitas imagens.
