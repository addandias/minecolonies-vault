# Capturas por construção

Crie uma pasta com o identificador oficial normalizado de cada construção. Exemplo:

```text
construcoes/
└── composter/
    ├── principal.png
    ├── itens-para-compostagem.png
    └── configuracoes.png
```

Consulte as regras de origem, versão e nomenclatura em [[assets/interfaces/README|Interfaces das construções]].
